# Command

<ul> 
    <li>/* */ - Комментарии в CSS</li>
    <li>!important - Присваивает максимальный приоритет</li>
    <li>backdrop-filter - Позволяет применить эффекты вроде размытия к области фонового рисунка, расположенной за элементом. Чтобы эффект стал заметен, сам элемент должен быть частично прозрачным</li>
    <li>backface-visibility - Определяет, показывать обратную сторону элемента или нет</li>
    <li>block-size - Определяет размер элемента по блочной оси</li>
    <li>direction: ltr | rtl - Предназначен для сайтов, в которых имеет значение направление текста</li>
    <li>
        <ul>all
            <li>initial - Изменить все свойства, применяемые к элементу, на исходные значения</li>  
            <li>inherit - Изменить все свойства, применяемые к элементу, на значения, наследуемые у родителя</li>
            <li>unset - Если свойство наследуется, то его значение принимается как inherit, в противном случае оно принимается как initial</li>     
        </ul>
    </li>
    <li>bottom: размер | проценты | auto - Устанавливает положение нижнего края содержимого элемента без учёта толщины рамок и отступов</li>
    <li>box-decoration-break: slice(эффекты оформления на месте «разреза» не отображаются) | clone(эффекты оформления на месте «разреза» отоюражаются) - Указывает, как оформлять фрагмент текста при его переносе на другую строку</li>
    <li>
        <ul>box-shadow:  - Добавляет тень к элементу
            <li>inset - Тень выводится внутри элемента</li>
            <li>Сдвиг по X - Обязательный. Смещение тени по горизонтали относительно элемента</li>
            <li>Сдвиг по Y - Обязательный. Смещение тени по вертикали относительно элемента</li>
            <li>Размытие - Задаёт радиус размытия тени</li>
            <li>Растяжение - Задает радиус растяжения</li>
            <li>Цвет - Задает цвет тени, в любом формате</li>
        </ul>
    </li>
    <li>
        <ul>box-sizing - Позволяет изменить алгоритм расчёта ширины и высоты элемента, чтобы свойства width и height задавали размеры не содержимого, а размеры блока
            <li>content-box - Основывается на стандартах CSS, при этом свойства width и height задают ширину и высоту содержимого и не включают в себя значения margin, padding и border</li>
            <li>border-box - Свойства width и height включают в себя значения padding и border, но не margin</li>
        </ul>
    </li>
    <li>caret-color: auto(подбирает под фон), цвет - Задаёт цвет текстового курсора, который отображается в текстовых полях формы</li>
    <li>float: left, right, none - определяет по какой стороне  будет выравниваться элемент, при этом остальные элементы будут обтекать его с других сторон</li>
    <li>clear: left, right, both(с двух сторон), none - отменяет действие float с указанной стороны для вышестоящих элементов</li>
    <li>color: red - Определяет цвет текста элемента</li>
    <li>cursor: url, auto, none - Устанавливает форму курсора, когда он находится в пределах элемента</li>
    <li>
        <ul>content: - Позволяет вставлять генерируемое содержание в текст веб-страницы, которое первоначально в тексте отсутствует. Применяется совместно с псевдоэлементами ::after и ::before, они соответственно указывают отображать новое содержимое после или до содержимого элемента, к которому добавляются
            <li>строка - Текст, который добавляется на веб-страницу, строка при этом должна браться в двойные или одинарные кавычки</li>
            <li>attr(атрибут) - Возвращает строку, которая является значением атрибута тега указанного в скобках</li>
            <li>open-quote - Вставляет открывающую кавычку, тип которой устанавливается с помощью стилевого свойства quotes</li>
            <li>close-quote - Вставляет закрывающую кавычку</li>
            <li>no-open-quote - Отменяет добавление открывающей кавычки</li>
            <li>no-close-quote - Отменяет добавление закрывающей кавычки</li>
            <li>url - Абсолютный или относительный адрес вставляемого объекта</li>
            <li>
                <ul>counter - Выводит значение счётчика
                    <li>counter-reset: переменная | число | none | inherit - устанавливает переменную, в которой будет храниться счётчик отображений определенного элемента, а также начальное значение счётчика</li>
                    <li>counter-increment: переменная | число | none - предназначено для увеличения значения счётчика приращений, который задается свойством counter-reset</li>
                </ul>
            </li>    
            <li>none - Не добавляет никакого содержимого</li>
            <li>normal - Задаётся как none для псевдоэлементов ::before и ::after</li>
        </ul>
    </li>
    <li>
        <ul>FLEX
            <li>
                <ul>align-content - Задаёт тип выравнивания строк внутри флекс-контейнера по поперечной оси при наличии свободного пространства
                    <li>stretch(по умолчанию) - строки равномерно растягиваются, заполняя всё свободное пространство</li>
                    <li>flex-start - Строки располагаются в начале поперечной оси. Каждая строка идёт вровень с предыдущей</li>
                    <li>center - Строки располагаются по центру</li>
                    <li>flex-end - Строки располагаются начиная с конца поперечной оси. Каждая предыдущая строка идёт вровень со следующей</li>
                    <li>space-between - Строки равномерно распределяются в контейнере и расстояние между ними одинаково</li>
                    <li>space-around - Строки равномерно распределяются таким образом, чтобы пространство между двумя соседними строками было одинаковым. Пустое пространство перед первой строкой и после последней строки равно половине пространства между двумя соседними строками</li>
                    <li>space-evenly - Строки равномерно распределяются таким образом, чтобы пространство между двумя соседними строками, а также пространство перед первой строкой и после последней строки было одинаковым</li>
                </ul>
            </li>
            <li>
                <ul>align-items - выравнивает флекс-элементы внутри контейнера в перпендикулярном направлении
                    <li>stretch(по умолчанию) - Флексы растягиваются таким образом, чтобы занять всё доступное пространство контейнера</li>
                    <li>flex-start - Флексы выравниваются в начале поперечной оси контейнера</li>
                    <li>center - Флексы выравниваются по линии поперечной оси</li>
                    <li>flex-end - Флексы выравниваются в конце поперечной оси контейнера</li>
                    <li>baseline - Флексы выравниваются по их базовой линии</li>
                </ul>
            </li>
            <li>
                <ul>align-self - выравнивает флекс-элементы текущей строки, переписывая значение align-items
                    <li>auto(по умолчанию) - Берёт родительское значение align-items или stretch, если нет родителя</li>
                    <li>flex-start - Берёт родительское значение align-items или stretch, если нет родителя</li>
                    <li>flex-end - Элемент выравнивается в конце поперечной оси контейнера</li>
                    <li>center - Элемент выравнивается по центральной линии на поперечной оси</li>
                    <li>baseline - Элемент выравнивается по базовой линии текста</li>
                    <li>stretch - Элемент растягивается таким образом, чтобы занять всё свободное пространство контейнера по поперечной оси</li>
                </ul>
            </li>
        </ul>
    </li>
    <li>
        <ul>ANIMATION
            <li>animation: name, duration, timing-function, delay, iteration-count, direction, fill-mode, play-state</li>
            <li>animation-delay: 2s - устанавливает время ожидания перед запуском цикла анимации</li>
            <li>    
                <ul>animation-direction - Устанавливает направление движения анимации
                    <li>normal - Анимация идёт с самого начала, после завершения цикла возвращается к исходному состоянию</li>
                    <li>alternate - Анимация идёт с начала до конца, затем плавно возвращается в исходное положение</li>
                    <li>reverse - Анимация идёт с конца цикла, после его завершения возвращается к исходному состоянию</li>
                    <li>alternate-reverse - Анимация идёт с конца до начала, затем плавно возвращается в исходное положение</li>
                </ul>
            </li>
            <li>animation-duration: 2s - Задаёт время в секундах или миллисекундах, сколько должен длиться один цикл анимации</li>
            <li>animation-iteration-count: infinite | число - Сколько раз должна повторяться анимация</li>  
            <li>    
                <ul>animation-fill-mode - Определяет, какие стили должны применяться к элементу, когда анимация не проигрывается
                    <li>none - К элементу не применяются какие-либо стили</li>
                    <li>forwards - К элементу по окончанию анимации применяется стиль последнего ключевого кадра</li>
                    <li>backwards - К элементу применяется стиль первого ключевого кадра и он остаётся на протяжении периода заданного animation-delay</li>
                    <li>both - К элементу применяются оба правила, как для forwards, так и для backwards</li>
                </ul>
            </li>
            <li>animation-name: none | Идентификатор - Устанавливает одну или несколько анимаций, которые применяются к элементу</li>
            <li>animation-play-state: running | paused - Свойство определяет, проигрывать анимацию или поставить её на паузу</li>
            <li>    
                <ul>animation-timing-function - Устанавливает, согласно какой функции времени должна происходить анимация каждого цикла между ключевыми кадрами
                    <li>ease - Анимация начинается медленно, затем ускоряется и к концу движения опять замедляется</li>
                    <li>ease-in - Анимация медленно начинается, к концу ускоряется</li>
                    <li>ease-out - Анимация начинается быстро, к концу замедляется</li>
                    <li>ease-in-out - Анимация начинается и заканчивается медленно</li>
                    <li>linear - Одинаковая скорость от начала и до конца</li>
                    <li>step-start - Как таковой анимации нет. Стилевые свойства сразу же принимают конечное значение</li>
                    <li>step-end - Как таковой анимации нет. Стилевые свойства находятся в начальном значении заданное время, затем сразу же принимают конечное значение</li>
                    <li>steps - Ступенчатая функция, имеющая заданное число шагов</li>
                    <li>cubic-bezier - Задаёт функцию движения в виде кривой Безье</li>
                </ul>
            </li>
        </ul>
    </li>
    <li>
        <ul>BACKGROUND
            <li>background: background-image, background-position, background-repeat, background-attachment, background-origin, background-clip, background-color - позволяет установить одновременно несколько характеристик фона</li>
            <li>
                <ul>background-attachment - устанавливает, будет ли прокручиваться фоновое изображение вместе с содержимым элемента
                    <li>fixed - Делает фоновое изображение элемента неподвижным</li>
                    <li>scroll - Позволяет перемещаться фону вместе с содержимым</li>
                    <li>local - Фон фиксируется с учётом поведения элемента. Если элемент имеет прокрутку, то фон будет прокручиваться вместе с содержимым, но фон, выходящий за рамки элемента, остаётся на месте</li>
                </ul>
            </li>
            <li>
                <ul>background-blend-mode - Определяет режим наложения фонового изображения на фоновый цвет или другие фоновые изображения
                    <li>normal - Обычный. Не используется смешивание цветов</li>
                    <li>multiply - Умножение. В этом режиме значение основного цвета умножается на значение совмещённого цвета. Результирующий цвет всегда представляет собой более тёмный цвет</li>
                    <li>screen - Осветление. В этом режиме перемножаются обратные значения основного и совмещённого цвета. В качестве результирующего цвета всегда применяется более светлый цвет</li>
                    <li>overlay - Перекрытие. В этом режиме цвета умножаются или осветляются в зависимости от основного цвета</li>
                    <li>darken - Замена тёмным. В качестве результирующего выбирается основной или совмещённый цвет, в зависимости от того, какой из них темнее</li>
                    <li>lighten - Замена светлым. В качестве результирующего цвета выбирается основной или совмещённый цвет, в зависимости от того, какой из них светлее</li>
                    <li>color-dodge - Осветление основы. Основной цвет заменяется более ярким для отражения совмещённого цвета в результате уменьшения контраста между двумя цветами</li>
                    <li>color-burn - Затемнение основы. Основной цвет заменяется более тёмным для отражения совмещённого цвета в результате увеличения контраста между двумя цветами</li>
                    <li>hard-light - Направленный свет. В этом режиме цвета умножаются или осветляются в зависимости от совмещённого цвета. Полученный эффект аналогичен освещению изображения резким светом прожектора</li>
                    <li>soft-light - Рассеянный свет. В этом режиме цвета становятся более тёмными или более светлыми в зависимости от совмещённого цвета. Полученный эффект аналогичен освещению изображения расплывчатым светом прожектора</li>
                    <li>difference - Разница. В этом режиме вычитается либо совмещённый цвет из основного цвета, либо основной цвет из совмещённого цвета в зависимости от того, какой цвет имеет большее значение яркости</li>
                    <li>exclusion - Исключение. В этом режиме создается эффект, аналогичный создаваемому в режиме «Разница», но характеризующийся более низким контрастом</li>
                    <li>hue - Тон. В этом режиме создаётся результирующий цвет с яркостью и насыщенностью основного цвета и цветовым тоном совмещённого цвета</li>
                    <li>saturation - Насыщенность. В этом режиме создаётся результирующий цвет с яркостью и цветовым тоном основного цвета и насыщенностью совмещённого цвета</li>
                    <li>color - Цвет. Создаётся результирующий цвет с яркостью основного цвета и с цветовым тоном и насыщенностью совмещённого цвета</li>
                    <li>luminosity - Яркость. В этом режиме создаётся результирующий цвет с цветовым тоном и насыщенностью основного цвета и яркостью совмещённого цвета. Этот режим создаёт эффект инверсии цвета</li>
                </ul>
            </li>
            <li>
                <ul>background-clip - Определяет, как цвет фона или фоновая картинка должна выводиться под границами
                    <li>padding-box - Фон отображается внутри границ</li>
                    <li>border-box - Фон выводится под границами</li>
                    <li>content-box - Фон отображается только внутри контента</li>
                    <li>text - Фон отображается только внутри текста</li>
                </ul>
            </li>
            <li>background-color: red; - Определяет цвет фона элемента</li>
            <li>background-image: url(/путь/img.jpg); - Устанавливает фоновое изображение для элемента</li>
            <li>
                <ul>background-origin - Определяет область позиционирования фонового рисунка
                    <li>padding-box - Фон позиционируется относительно края элемента с учетом толщины границы</li>
                    <li>border-box - Фон позиционируется относительно границы, при этом линия границы может перекрывать изображение</li>
                    <li>content-box - Фон позиционируется относительно содержимого элемента</li>
                </ul>
            </li>
            <li>
                <ul>background-position: ось X ось Y - Задаёт начальное положение фонового изображения
                    <li>background-position-x: left | center | right | проценты | размер - Задаёт положение фонового изображения внутри элемента по горизонтали</li>
                    <li>background-position-y: top | center | bottom | проценты | размер - Задаёт положение фонового изображения внутри элемента по вертикали</li>
                </ul>
            </li>
            <li>
                <ul>background-repeat: Определяет, как будет повторяться фоновое изображение
                    <li>no-repeat - Устанавливает одно фоновое изображение в элементе без его повторений</li>
                    <li>repeat - Фоновое изображение повторяется по горизонтали и вертикали</li>
                    <li>repeat-x - Фоновый рисунок повторяется только по горизонтали</li>
                    <li>repeat-y - Фоновый рисунок повторяется только по вертикали</li>
                    <li>space - Изображение повторяется столько раз, чтобы полностью заполнить область; если это не удаётся, между картинками добавляется пустое пространство</li>
                    <li>round - Изображение повторяется так, чтобы в области поместилось целое число рисунков; если это не удаётся сделать, то фоновые рисунки масштабируются</li>
                </ul>
            </li>
            <li>
                <ul>background-size: Масштабирует фоновое изображение, согласно заданным размерам
                    <li>размер - Задаёт размер в любых доступных для CSS единицах</li>
                    <li>проценты - Задаёт размер фоновой картинки в процентах от ширины или высоты элемента</li>
                    <li>auto - Если задано одновременно для ширины и высоты (auto auto), размеры фона остаются исходными</li>
                    <li>cover - Масштабирует изображение с сохранением пропорций так, чтобы его ширина или высота равнялась ширине или высоте блока</li>
                    <li>contain - Масштабирует изображение с сохранением пропорций таким образом, чтобы картинка целиком поместилась внутрь блока</li>
                </ul>
            </li>
        </ul>
    </li>
    <li>
        <ul>BORDER
            <li>border: border-width, border-style, border-color - позволяет одновременно установить толщину, стиль и цвет границы вокруг элемента</li>
            <li>border-width: размер | thin(2px) | medium(4px) | thick(6px) - Устанавливает толщину границы вокруг элемента</li>
            <li>
                <ul>border-style - Устанавливает стиль границы вокруг элемента
                    <li>none - Линия не отображается и значение её толщины обнуляется</li>
                    <li>hidden - Имеет тот же эффект, что и none за исключением применения border-bottom-style к ячейкам таблицы, у которой значение свойства border-collapse установлено как collapse</li>
                    <li>dotted - Линия состоящая из набора точек</li>
                    <li>dashed - Пунктирная линия, состоящая из серии коротких отрезков</li>
                    <li>solid - Сплошная линия</li>
